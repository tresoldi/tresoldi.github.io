I write open-source tools for computational linguistics and data analysis. Everything is on
[GitHub](https://github.com/tresoldi); most Python packages install with a plain
`pip install <name>`. Most of these started because I needed the thing and it didn't exist,
or existed in a shape I couldn't use at six in the morning over a phone's SSH connection.
They are open because research you can't re-run isn't really research — and because other
people's open code is how I learned.
