I write open-source tools for computational linguistics and data analysis. Everything is
on [GitHub](https://github.com/tresoldi); most Python packages install with a plain
`pip install <name>`. The guiding principles are simple — open by default, documented,
tested, and built to make reproducible research a little easier.
